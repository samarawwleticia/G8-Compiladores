# Sobre

Esse repositório foi criado com o objetivo de desenvolver um compilador que realiza a tradução da linguagem **C** para a linguagem **Lua**, aplicando as análises léxicas, sintáticas e semânticas estudadas na disciplina de Compiladores 1, com a orientação do docente **Sergio Antônio Andrade de Freitas**.

O compilador não aborda todas as funcionalidades da linguagem C ou Lua, visto que foi simplificado e criado apenas para aprendizado prático da disciplina.
